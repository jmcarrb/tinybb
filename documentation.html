Documentation

Installation Instructions

To install TinyBB 1.4, upload all of the files to your web server, then access your phpMyAdmin, create a database and import "TinyBB Database.sql". After this, you can edit your config.php file located in the forum root directory and edit your database details.

Settings such as your bulletin board title can be editied using the administration panel.

Default Login Information

Username: admin
Password: tinybb123

It's best to change the password after logging in for security reasons.

Editing Themes

Theme editing is fairly simple, to edit the main theme, edit index.php (root directory) and style/style.css.

Modification Installation

Most official mods from TinyBB.net will contain a detailed documentation file to help you install the modification, however most are simple with a 1 line template edit and a file upload to forum/mods

User Ranks

In this version (1.4) there are 3 groups, Member & Admin, Moderator and user level 3 for banned. If you're setting a user to admin via the database, the admin ID they need is 1, 2 will be for moderators however they have no priverlages at this time.
